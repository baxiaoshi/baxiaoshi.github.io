发表于 2018-09-05 15:37   |   分类于 k8s ， k8s那些天

Request与RequestInfoFactory

k8s中由apiserver负责所有资源的增、删、改、查、Watch的操作，为了实现对所有资源上层的统一操作，包含：身份验证、权限检查、跨域、日志、超时等，为了实现对各种资源的统一操作，apiserver会将每一个http请求进行解析，放入到一个RequestInfo中

  • RequestInfo

  • RequestInfoFactory: 从原始的http.Request中解析出上面的RequestInfo

filter

对于每一个http请求都需要做一些类似身份认证、请求格式化、记录等操作，apiserver中通过filters来进行统一进行处理, 基础的filters主要由DefaultBuildHandlerChain来进行定义
